# Python-Calculator
A basic calculator I coded using Python, mainly to just learn and make my first project.
It can perform addition (+), subtraction (-), multiplication (*), division (/), modulus (%), and exponentiation (**).
Some of the text and code may be in Indonesian.
I plan to include more complex calculations in the future, like square roots, trigonometric functions, and memory functions.

To use the code and run it, just copy it into your editor, or download the file and open it on your editor.
After that, run it, then use it, it already explains most of the calculations it can do via text.
